generate(self, number=1, size=5, target_size=5, legal_target_size=10, VOTERS=1000, OBJECTS=200, bad_hideout=False, slice_level=0)

source code 

Metoda generuje dane głosowania (listę obiektów wraz z indetyfikatorami głosujących, którzy oddali głos na dany obiekt) i zapisuje je do pliku określonego w self.path.

Parameters:
  • number - liczba klik
  • size - rozmiar kliki
  • target_size - liczba obiektów, na które głosują członkowie kliki
  • legal_target_size - średnie liczba obiektów, na które głosują uczciwi głosujący
  • VOTERS - liczba głosujących
  • OBJECTS - liczba obiektów
  • bad_hideout (boolean) - parametr mówu o tym, czy członkowie kliki nie głosują na obiekty nie będące ich zadanym celem.
